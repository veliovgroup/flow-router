### setQueryParams method

```js
FlowRouter.setQueryParams(qs);
```

- `qs` {*String*} - Serialized query string, `{ key: 'val' }`
- Returns {*true*}

#### Unset parameter

To remove a query param set it to `null`:

```js
FlowRouter.setQueryParams({ paramToRemove: null });
```
